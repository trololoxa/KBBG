+===============================================РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ===============================================+
Назначение игры
    Игра "Художественный поединок" создана для тренировки навыков написания курсового проекта в C++ с использованием
    принципов объектно-ориентированного программирования (ООП). Игра развивает у игроков навыки планирования и
    стратегического мышления: чтобы выжить как можно дольше, игрок должен эффективно управлять ресурсами в виде
    картин и монет.

Целевая аудитория
    Программисты, желающие ознакомиться с исходным кодом проекта.
    Любители игр в жанре "Rogue-like".
    Все, кто заинтересован в улучшении своих навыков планирования.
Системные требования
    Операционная система: Microsoft Windows 7 и выше.
    Архитектура процессора: Core 2 Duo E6300 и выше.
    Объем памяти: от 1 до 4 МБ.
    Требования к оборудованию: клавиатура для ввода.
Интерфейс
    Основной экран игры - консольное окно Windows.
    Управление осуществляется с помощью клавиатуры.
    Для каждого этапа и каждой локации предусмотрен свой набор действий, отображаемых в меню.
Этапы игры
    Первый этап: Выбор локации
        Игрок оказывается между двумя дверями:
            Первая дверь: название локации отображается.
            Вторая дверь: название локации неизвестно.
        Действия на этом этапе
            Выбор двери: переход на соответствующую локацию.
            Сохранение прогресса: игрок может сохранить текущее состояние игры.
    Второй этап: Игровой процесс
        После выбора одной из дверей игрок попадает на одну из трёх локаций, на которых уже происходит игровой процесс.
        После выхода с локации игрок попадает обратно на первый этап.
Ограничения
    Игрок не может выполнять действия, не указанные в меню локации.
